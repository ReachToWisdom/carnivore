# 3장. 조리법과 주간 식단

---

## 1. 기본 조리법

### 완벽한 스테이크 (팬 시어링)

카니보어의 기본이자 핵심 조리법이다.

```
재료: 소고기 등심 또는 안심 200~300g, 소금, 버터

1. 고기를 냉장고에서 30분 전에 꺼내 실온에 둔다
2. 키친타올로 표면 수분을 완전히 제거한다
3. 소금을 양면에 넉넉히 뿌린다
4. 팬을 강불로 달궈 연기가 살짝 날 때까지 가열한다
5. 소기름 또는 버터를 두르고 고기를 올린다
6. 한 면당 2~3분씩 구운다 (미디엄 레어 기준)
7. 불을 끄고 버터 한 조각을 올려 1분 레스팅한다
8. 3분간 도마 위에서 레스팅 후 썰어 먹는다
```

> **카니보어 전문가 팁:**
> 익힘 정도는 개인 취향이지만,
> 미디엄 레어~미디엄이 영양소 보존에 가장 좋다.
> 과도한 가열(웰던)은 단백질 변성과 지방 산화를 일으킨다.

### 뼈국물 (본 브로스)

장 건강 회복의 핵심이자 전해질 보충의 최고 수단이다.

```
재료: 사골 또는 소꼬리 1~1.5kg, 물 3L, 소금, 식초 1큰술

1. 사골을 찬물에 1시간 담가 핏물을 뺀다
2. 끓는 물에 5분간 데쳐 불순물을 제거한다
3. 깨끗한 물 3L에 사골과 식초 1큰술을 넣는다
   (식초가 뼈에서 미네랄을 추출하는 데 도움)
4. 강불로 끓인 뒤 약불로 줄여 12~24시간 끓인다
5. 식힌 후 위의 기름층을 걷어 따로 보관한다
   (이 기름도 훌륭한 조리용 지방이다)
6. 소금으로 간을 하고 컵에 따라 마신다
```

> **기능의학 전문가 관점:**
> 뼈국물의 글리신은 간 해독(phase II conjugation)을 지원하고,
> 콜라겐은 장 점막의 타이트 정션 회복을 돕는다.
> 적응기에 하루 1~2잔을 권장하는 이유다.

### 소간 요리 — 입문자용

```
재료: 소간 150g, 버터 30g, 소금

1. 소간을 우유(또는 물)에 30분 담가 비린 맛을 줄인다
2. 키친타올로 물기를 제거하고 5mm 두께로 썬다
3. 소금을 뿌린다
4. 팬에 버터를 녹이고 중불에서 한 면당 1~2분 구운다
   (과하게 익히면 질겨진다 — 살짝 핑크빛이 남을 정도)
5. 버터를 끼얹으며 마무리한다
```

맛이 부담스러운 사람은:
- 다진 소고기에 소간을 20% 비율로 섞어 패티를 만든다
- 얼린 소간을 갈아서 캡슐에 넣어 먹는다 (비타민 대체)

## 3. 주간 식단 예시

### 1인 가구 — Level 1 (첫 30일)

| 요일 | 아침 (또는 첫 끼) | 저녁 (또는 둘째 끼) |
|------|-----------------|-------------------|
| 월 | 소금 스테이크 200g + 뼈국물 1잔 | 삼겹살 200g |
| 화 | 차돌박이 150g | 등심 스테이크 250g + 버터 |
| 수 | 갈비 200g | 사골곰탕 (고기 건더기 포함) |
| 목 | 등심 200g + 뼈국물 1잔 | 양지 수육 200g |
| 금 | 삼겹살 200g | 소금 스테이크 250g |
| 토 | 차돌박이 200g | 갈비찜 (소금 간만) |
| 일 | 뼈국물 + 소고기 | 등심 스테이크 200g + 버터 |

### 2인 가구 — Level 2 (확장기)

| 요일 | 첫 끼 | 둘째 끼 |
|------|-------|--------|
| 월 | 스크램블 에그 4개 + 버터 | 등심 스테이크 + 뼈국물 |
| 화 | 연어 팬구이 + 달걀 | 삼겹살 + 소간 패티 |
| 수 | 사골곰탕 + 수란 | 갈비 구이 |
| 목 | 차돌박이 + 달걀 | 고등어 구이 + 뼈국물 |
| 금 | 소금 스테이크 + 버터 | 소꼬리찜 |
| 토 | 골수 로스팅 + 달걀 | 한우 등심 + 연어 |
| 일 | 브런치: 에그+삼겹+버터 | 사골곰탕 + 소고기 |

## 4. 외식·회식 대처법

카니보어 식단에서 가장 현실적인 난관이 외식과 회식이다.
완벽하지 않아도 된다. **최선의 선택을 하면 된다.**

| 상황 | 추천 메뉴 | 주의 사항 |
|------|----------|----------|
| **고깃집** | 소고기 구이, 삼겹살 | 쌈장·마늘·양념 대신 소금 |
| **한식당** | 설렁탕, 갈비탕, 소머리국밥 | 밥은 남기고 국물+고기만 |
| **일식** | 사시미, 연어, 장어 | 간장·와사비 소량 OK |
| **양식** | 스테이크 단품 | 빵·감자 사이드 빼달라고 요청 |
| **회식** | 고깃집 추천 | 술은 최소화, 소주보다 위스키(당 없음) |
| **편의점** | 삶은 달걀, 소고기 편육 | 성분표 확인 (첨가물 주의) |

> **실전 팁:** "요즘 탄수화물을 줄이고 있어요"라고 하면
> 대부분의 상황에서 무리 없이 넘어간다.
> 카니보어를 설명할 필요는 없다.

## 5. 예산별 식단 구성

### 저예산 (월 30~50만 원)

- 수입 소고기 (미국산·호주산) 위주
- 삼겹살, 목살 활용
- 계란 (하루 4~6개)
- 사골 (대량 구매 후 냉동)
- 소간, 소심장 (저렴하면서 영양 최고)

### 중예산 (월 50~80만 원)

- 한우 + 수입산 혼합
- 연어, 고등어 추가
- 목초 사육 버터
- 내장육 정기 구매

### 고예산 (월 80만 원 이상)

- 목초 사육 한우·수입 소고기
- 자연산 연어, 굴
- 기(Ghee), 목초 버터
- 목초 사육 소간·골수

> **핵심:** 예산이 적어도 카니보어는 가능하다.
> 수입 소고기 + 계란 + 삼겹살 + 사골이면
> 영양학적으로 어떤 채소 중심 식단보다 완전하다.
