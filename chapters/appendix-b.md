# 부록 D-1. 실전 혈액검사 해석 가이드

---

카니보어·저탄고지 식단 전환 후 혈액검사 수치가 기존 기준과 달라질 수 있다.
아래 표는 주요 검사 항목의 해석 방법을 정리한 것이다.

## 주요 검사 항목 해석

| 검사 항목 | 이상적 범위 | 카니보어 후 변화 | 해석 |
|----------|-----------|----------------|------|
| 총 콜레스테롤 | — | 상승 가능 | 단독 수치로 의미 없음. 입자 구성이 중요 |
| LDL-C | — | 상승 가능 | 입자 크기가 핵심: 크고 가벼운 lbLDL(양성) vs 작고 밀도 높은 sdLDL(위험) |
| HDL-C | > 50 mg/dL | 상승 | 심혈관 보호 지표. 높을수록 좋음 |
| 중성지방(TG) | < 100 mg/dL | 하락 | 인슐린 민감도 개선 신호 |
| **TG/HDL 비율** | **< 2.0** | **개선** | **가장 신뢰할 수 있는 심혈관 위험 지표** |
| 공복혈당 | < 100 mg/dL | 하락 또는 약간 상승 | 상승 시 "생리적 인슐린 저항성" 가능(병적 아님) |
| HbA1c | < 5.7% | 개선 | 3개월 평균 혈당. 공복혈당보다 정확 |
| 공복 인슐린 | < 5 μIU/mL | 하락 | 대사 건강의 핵심 지표 |
| hs-CRP | < 1.0 mg/L | 하락 | 전신 염증 감소 |
| 요산 | 남 3.5~7.2 | 초기 상승 → 정상화 | 적응기 현상. 수분 섭취로 완화 |
| GGT | < 30 U/L | 하락 | 간 건강 개선·산화 스트레스 감소 |
| 페리틴 | 50~150 ng/mL | 상승 가능 | 철분 저장. 염증 배제 후 해석 |

## 의사에게 요청할 추가 검사

일반 건강검진에 포함되지 않지만 식단 효과를 정확히 평가하려면 아래 검사를 요청한다.

| 추가 검사 | 목적 | 비고 |
|----------|------|------|
| NMR LipoProfile | LDL 입자 수·크기 측정 | sdLDL 비율 확인 |
| 공복 인슐린 | 인슐린 저항성 직접 평가 | HbA1c와 함께 해석 |
| CAC Score (관상동맥 칼슘) | 실제 동맥경화 정도 | LDL 높아도 CAC 0이면 저위험 |
| Lp(a) | 유전적 심혈관 위험 | 식단과 무관, 1회 측정 |
| 호모시스테인 | B12·엽산 대사 지표 | 카니보어 시 보통 개선 |

## 의사와 대화하는 법

> "LDL이 올랐다"는 말에 당황하지 말 것. 아래 세 가지를 확인하라:
> 1. **TG/HDL 비율이 2 미만인가?** → 맞다면 LDL 입자 대부분이 양성(lbLDL)
> 2. **NMR 검사로 입자 크기를 확인했는가?** → 미확인 시 요청
> 3. **CAC Score가 0인가?** → 0이면 실질적 동맥경화 없음

## 검사 시기 권장

| 시점 | 목적 |
|------|------|
| 식단 시작 전 | 기저선(baseline) 확보 |
| 3개월 후 | 초기 변화 확인, 적응기 수치 구분 |
| 6개월 후 | 안정화된 수치 확인 |
| 이후 연 1회 | 장기 추적 |
